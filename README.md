# Diabetes Prediction using Explainable AI (XAI)

## Project Overview

This project aims to predict diabetes using machine learning and deep learning models combined with Explainable AI (XAI) techniques. The focus is on achieving reliable predictions while ensuring transparency and interpretability for healthcare applications.



## Objectives

* Predict diabetes using healthcare indicators
* Apply multiple ML/DL models
* Interpret model decisions using XAI techniques (SHAP)
* Compare performance across different models



## Dataset

* **Dataset:** BRFSS 2015 Diabetes Health Indicators
* **Type:** Tabular healthcare dataset
* **Size:** +70,000 samples after preprocessing
* **Features:** Health, demographic, and lifestyle indicators (BMI, Age, Blood Pressure, etc.)
